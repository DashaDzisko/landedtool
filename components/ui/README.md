# UI primitives

Radix-based primitives styled with Linear tokens from `app/globals.css`.

| Component | Use |
|-----------|-----|
| `dialog` | Modals — add/edit application |
| `dropdown-menu` | Card actions, user menu |
| `select` | Status filter, sort, form fields |
| `tabs` | Settings panels |
| `toast` | Save confirmations, errors |
| `separator` | Section dividers |

Import from `@/components/ui` or individual files. Product components live in `atoms/`, `molecules/`, `organisms/`.
